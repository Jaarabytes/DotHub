# ProcastiMate


We help you procastinate your tasks more efficiently.

## Installation

Prerequisites:
- Node and Npm installed
- API Keys, described in the `.env.example` file

Run the following commands:
```
git clone git@github.com:Jaarabytes/pro-castination.git
cd procastimate
npm install 
npm run dev
```
